# Adicionando sua foto

Para adicionar sua foto pessoal ao FeedPlanner:

1. **Prepare sua foto:**
   - Formato recomendado: JPG ou PNG
   - Dimensões ideais: 400x400 pixels (quadrada)
   - Tamanho máximo: 500KB
   - Qualidade boa para exibição web

2. **Adicione ao projeto:**
   - Salve sua foto nesta pasta como `henrique-photo.jpg`
   - Ou renomeie o arquivo no código se usar um nome diferente

3. **Formatos aceitos:**
   - `henrique-photo.jpg` (recomendado)
   - `henrique-photo.png`
   - `henrique-photo.jpeg`

4. **Caso não adicione foto:**
   - O placeholder com ícone será exibido automaticamente
   - Você pode personalizar o placeholder editando o arquivo `Sobre.js`

**Localização do arquivo:** `client/public/images/henrique-photo.jpg`

**Exemplo de como ficará:**
- Foto circular de 300x300px na desktop
- Foto circular de 200x200px no mobile
- Com sombra e bordas arredondadas
- Integrada ao design responsivo

**Dica:** Para melhores resultados, use uma foto com fundo neutro e seu rosto bem centralizado.
